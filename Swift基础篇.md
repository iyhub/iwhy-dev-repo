# Swift变量

## 声明变量

```swift
let number : Phone = "138000138000"
```



```swift
var number : Phone = "138000138000"
```





## 字符串拼接

```swift
var greeting = "Hello, playground"
print(greeting)

var output:String = "im iy,\(greeting)"

print(output)
```

## 自动类型推断

```swift

```

## Decimal类型声明

```swift
let d2:Decimal = 3.24
print(d2)
// 3.240000000000000512

let d3 = Decimal(string:"3.24")!
print(d3)
// 3.24
```
